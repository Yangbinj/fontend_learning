CSS 定位 (Positioning) 属性允许你对元素进行定位。

fixed 是用来相对于body进行绝对定位的,元素框从文档流完全删除，并相对于其包含块定位。

this is fix position
hello

relative 是用来进行相对定位的，相对定位实际上被看作普通流定位模型的一部分，因为元素的位置相对于它在普通流中的位置。

this is relative position
hello1 hello2

position:absolute；绝对定位使用通常是父级定义position:relative定位，子级定义position:absolute绝对定位属性，并且子级使用left或right和top或bottom进行绝对定位。

hello

absolute 定位的元素脱离了文档流

hello ,absolute2,这个紧贴在p标签，证明absolute脱离文档流
